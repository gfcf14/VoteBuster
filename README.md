VoteBuster
----------

A C# based XNA shooter game, which includes use of timers, object movement, and collision detection. Please go to https://www.greendreampg.com/Games/votebuster.php in order to see this game in action!

NOTE: This repository only includes the main classes used in its C# programming: VoteBuster.cs and Vote.cs. As this game has now been transformed to JavaScript, you may see its internal code on the game's website above, by clicking on "inspect" or viewing the web page's source.
